メンバー名

あなた専用のカードを表示・保存するための名前です。スプレッドシートの member 列と同じ名前を半角英小文字で入力してください（例：naoya）。この端末のブラウザにのみ保存されます。

GAS の外部接続権限が未設定です 例文生成には Apps Script の承認が必要です（APIキーとは別設定です）。
  1. EnglishPhrase スプレッドシートを開く
  2. 拡張機能 → Apps Script → 関数 authorizeExternalRequest を実行
  3. 表示された「権限を確認」→ 自分の Google アカウントで許可
  4. デプロイ → デプロイを管理 → 新バージョン（実行者: 自分 / アクセス: 全員）
APIキー設定

例文生成に Anthropic の API キーを使います。キーはこの端末のブラウザ（localStorage）にのみ保存され、保存のたびに GAS 経由で送信されます。 キーを取得する →
